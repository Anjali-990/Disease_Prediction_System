About Our System

An AI-powered system designed to assist users in identifying possible diseases based on symptoms.

Project Overview

The Disease Prediction System is an intelligent web application that predicts possible diseases based on symptoms entered by users. The system uses Machine Learning algorithms to analyze symptom data and provide predictions along with confidence scores.

The goal of this project is to demonstrate how Artificial Intelligence can assist in preliminary health analysis and help users understand potential health conditions before consulting medical professionals.

Also, System have its Dataset limitations. Hence, its a PROBABILISITC MODEL, not a diagnostic tool.

Key Features

🧠 AI Prediction

Predicts diseases using machine learning models.

📊 Confidence Score

Displays prediction confidence percentage.

📜 Prediction History

Stores user predictions for future reference.

📈 Admin Dashboard

Visual analytics of diseases and trends.
